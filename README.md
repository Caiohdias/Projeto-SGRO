# 🌱 SGRO - Sistema de Gerenciamento de Resíduos Orgânicos


## 📖 Sobre o Projeto

O SGRO (Sistema de Gerenciamento de Resíduos Orgânicos) é uma solução desenvolvida para conectar supermercados e usinas de biogás, permitindo o gerenciamento completo do descarte de resíduos orgânicos de forma sustentável.

O sistema digitaliza todo o processo de geração, coleta e destinação dos resíduos, garantindo rastreabilidade, redução de erros operacionais e incentivo à economia circular.

Ao invés de descartar alimentos vencidos ou impróprios para consumo de maneira inadequada, os supermercados podem disponibilizar esses resíduos para usinas que transformam o material em energia limpa através da biodigestão.

## 🚀 Funcionalidades

### Supermercado

- Login no sistema
- Registro de resíduos orgânicos
- Cadastro de peso estimado
- Descrição dos resíduos
- Escolha entre coleta manual ou automática
- Geração de chamados de coleta
- Histórico completo de chamadas
- Acompanhamento do status das coletas

### Usina de Biogás

- Login no sistema
- Visualização dos chamados disponíveis
- Aceitação de coletas
- Controle de coletas em andamento
- Confirmação da coleta realizada
- Registro do peso real coletado
- Cálculo automático de energia gerada
- Cálculo automático do valor a ser pago ao supermercado
- Finalização do processo

---

## 🔄 Fluxo do Sistema

1. O supermercado registra os resíduos.
2. O sistema cria um chamado de coleta.
3. A usina visualiza os chamados disponíveis.
4. A usina aceita a coleta.
5. A coleta é realizada.
6. O peso real é registrado.
7. O sistema calcula:
   - Energia gerada (kWh)
   - Valor do pagamento
8. O chamado é finalizado.

## 🌎 Problema Resolvido

Segundo estudos sobre resíduos orgânicos, grande parte dos alimentos descartados por supermercados poderia receber destinação mais sustentável.

O SGRO foi criado para:

- Reduzir desperdícios
- Evitar descarte irregular
- Incentivar energia limpa
- Facilitar o controle operacional
- Promover sustentabilidade empresarial
